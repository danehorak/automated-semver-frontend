Testing automated semantic versioning with Django REST API and Angular

About This Project

This is a test project demonstrating automated semantic versioning using:

  • Backend: Django REST Framework with python-semantic-release
  • Frontend: Angular with semantic-release

Both repositories use GitHub Actions to automatically version releases based on conventional commits.

API Endpoints

The backend provides simple Widget and Cog REST endpoints for testing:

  • GET /api/widgets/ - List widgets
  • GET /api/cogs/ - List cogs
  • GET /api/version/ - Get API version
